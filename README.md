# 🛒 NexCart - Full Stack E-Commerce Application

A complete e-commerce platform built with the MERN Stack
(MongoDB, Express.js, React.js, Node.js)

## 🔗 Live Links

- **Live Demo:** https://nex-cart-hazel.vercel.app
- **Backend API:** https://nexcart-api-9t4z.onrender.com
- **GitHub:** https://github.com/HarikaMuthyala06/NexCart

## ✨ Features

### User Features

- User Registration and Login
- JWT Authentication
- Browse Products with Search and Filter
- Add to Cart and Update Quantity
- Place Orders with COD payment
- View Order History
- Edit Profile

### Admin Features

- Admin Dashboard with Stats
- Add, Edit, Delete Products
- View and Update Order Status
- View All Users

## 🛠️ Tech Stack

### Frontend

- React.js (Vite)
- Tailwind CSS
- React Router DOM
- Axios

### Backend

- Node.js
- Express.js
- MongoDB + Mongoose
- JWT Authentication
- bcryptjs

### Deployment

- Frontend → Vercel
- Backend → Render
- Database → MongoDB Atlas
